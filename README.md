Install
--------

Place `.xml` files into `share/qtcreator/styles` or `~/.config/QtProject/qtcreator/styles/` folder.

Compatibility
-------------

For Qt Creator 3.0 and below, remove line which starts with `<style name="PrimitiveType"` in `.xml`.
